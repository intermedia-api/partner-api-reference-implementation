# Intermedia Partner API samples

This repository provides links to reference implementations of Intermedia Public API in various languages:

* C# implementation: [https://github.com/intermedia-api/partner-api-sample-csharp](https://github.com/intermedia-api/partner-api-sample-csharp). 
This implementation showcases two distinct approaches to using the APIs: through a client generated by swagger-codegen, and through 'raw' HTTP calls. The authentication implementation also demonstrates the usage of ThinkTecture OAUTH client and 'raw' HTTP calls.
